Types of data analytics
- Descriptive analytics
- Diagnostic analytics
- Predictive analytics
- Prescriptive analytics

Descriptive analytics
- This is the simplest form of data analytics. 
- It is used to describe what has happened in the past. 
- It is used to summarize data and make it more understandable. 
- It is used to answer the question "What happened?".

Diagnostic analytics
- This is used to understand why something happened.
- It is used to answer the question "Why did it happen?".

Predictive analytics
- This is used to predict what will happen in the future.
- It is used to answer the question "What will happen?".

Prescriptive analytics
- This is used to recommend actions that can be taken to achieve a desired outcome.
- It is used to answer the question "What should we do?".

Python for data analytics
- Python is a programming language that is widely used for data analytics.
- It is open source and free to use.
- It has a large number of libraries that can be used for data analytics.
- Some of the popular libraries for data analytics in Python are pandas, numpy, matplotlib, seaborn, scikit-learn, etc.

Pandas
- Pandas is a library in Python that is used for data manipulation and analysis.
- It provides data structures like Series and DataFrame that make it easy to work with data.
- It provides functions for reading and writing data from different sources like CSV, Excel, SQL, etc.
- It provides functions for cleaning and transforming data.
- It provides functions for summarizing and visualizing data.

Numpy
- Numpy is a library in Python that is used for numerical computing.
- It provides data structures like arrays that make it easy to work with numerical data.
- It provides functions for performing mathematical operations on arrays.
- It provides functions for generating random numbers.
- It provides functions for linear algebra, Fourier transforms, etc.

Matplotlib
- Matplotlib is a library in Python that is used for data visualization.
- It provides functions for creating different types of plots like line plots, bar plots, scatter plots, etc.
- It provides functions for customizing the appearance of plots like colors, labels, titles, etc.
- It provides functions for saving plots to different file formats like PNG, PDF, etc.
